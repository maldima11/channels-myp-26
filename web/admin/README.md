# NUST Yield Portal - Admin Credentials Center (`/web/admin`)

This directory houses the administration panel, located inside the `/web` folder to ensure it is hosted as a single Vercel deployment.

---

## Features
1. **User Creation & Deletion**: Add and delete user login credentials for Farmers and Agritex Officers.
2. **Access Security**: Credentials list is synchronized using a shared browser `localStorage` key.
3. **Theme Toggling**: Supports Dark and Light themes, syncing the state instantly.
4. **Direct Navigation**: Link in the header redirects users back to the Yield Portal.
